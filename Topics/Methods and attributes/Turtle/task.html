Turtle

Here's a class Turtle that represents a turtle that is moving in the 2D world. The turtle can move in four directions: up, down, left and right. The turtle can make n steps at a time and the only restrictions are that \(x \geq 0, y \geq 0\).

class Turtle:
    def __init__(self, x, y):
        # the initial coordinates of the turtle
        self.x = x
        self.y = y

    def move_up(self, n):
        self.y += n

    def move_down(self, n):
        self.y = 0 if n > self.y else self.y - n

    def move_right(self, n):
        self.x += n

    def move_left(self, n):
        self.x = 0 if n > self.x else self.x - n
